# 杜邦分析框架详解

## 概述

杜邦分析（DuPont Analysis）是一种用于分解和评估公司盈利能力的财务分析方法。它将股东权益回报率（ROE）分解为三个关键组成部分，帮助投资者理解公司盈利能力的驱动因素。

## ROE 三因素分解

```
ROE = 净利率 × 资产周转率 × 权益乘数

其中：
- 净利率 = 净利润 / 营业收入
- 资产周转率 = 营业收入 / 总资产
- 权益乘数 = 总资产 / 股东权益
```

## 三因素详解

### 1. 净利率（Profit Margin）- 盈利能力

**定义**：每一元收入能产生多少净利润

**分析维度**：
- **毛利率**：反映产品定价能力和成本控制
- **期间费用率**：
  - 销售费用率：市场推广效率
  - 管理费用率：管理效率
  - 研发费用率：创新投入
  - 财务费用率：融资成本
- **净利率趋势**：盈利能力变化

**与商业模式的关系**：
- **高净利率**（>20%）：通常是品牌溢价、技术垄断、平台模式
  - 例：茅台（品牌）、腾讯（平台）、恒瑞医药（技术）
- **中等净利率**（10-20%）：有一定竞争优势
- **低净利率**（<10%）：薄利多销、规模经济模式
  - 例：零售、制造业

### 2. 资产周转率（Asset Turnover）- 运营效率

**定义**：每一元资产能产生多少收入

**分析维度**：
- **应收账款周转率**：收款效率，反映议价能力
- **存货周转率**：库存管理效率
- **固定资产周转率**：重资产利用效率
- **总资产周转率**：整体资产使用效率

**与商业模式的关系**：
- **高周转率**（>1.5）：轻资产、快周转模式
  - 例：零售、互联网、服务业
- **低周转率**（<0.5）：重资产模式
  - 例：房地产、制造业、基础设施

### 3. 权益乘数（Equity Multiplier）- 财务杠杆

**定义**：每一元股东权益支撑多少资产

**分析维度**：
- **资产负债率**：负债占总资产比例
- **有息负债率**：需要付息的负债比例
- **流动比率/速动比率**：短期偿债能力
- **利息保障倍数**：盈利对利息的覆盖

**与商业模式的关系**：
- **高杠杆**（>3）：金融、地产等资本密集型
  - 风险：利率敏感、债务压力
  - 优势：放大收益
- **低杠杆**（<2）：轻资产、现金流好的企业
  - 优势：财务稳健
  - 劣势：资本利用效率可能不足

## 典型商业模式的 ROE 特征

### 1. 品牌消费品（如茅台）
- **高净利率**：品牌溢价
- **中等周转率**：不需要快速周转
- **低杠杆**：现金流好，不需要高负债
- **ROE 驱动**：主要靠净利率

### 2. 互联网平台（如腾讯）
- **高净利率**：边际成本低
- **高周转率**：轻资产
- **低杠杆**：现金充裕
- **ROE 驱动**：净利率 + 周转率

### 3. 零售连锁（如永辉超市）
- **低净利率**：薄利多销
- **高周转率**：快速周转
- **中等杠杆**：应付账款
- **ROE 驱动**：主要靠周转率

### 4. 房地产（如万科）
- **中等净利率**：10-20%
- **低周转率**：开发周期长
- **高杠杆**：高负债经营
- **ROE 驱动**：主要靠杠杆

### 5. 银行
- **低净利率**：息差收入
- **极低周转率**：资产规模大
- **极高杠杆**：存款是负债
- **ROE 驱动**：主要靠杠杆

## 分析步骤

### 第一步：计算 ROE 及三因素
从财务报表中提取数据，计算5年趋势

### 第二步：横向对比
与同行业公司对比，找出差异

### 第三步：纵向分析
分析历史趋势，识别变化

### 第四步：结合商业模式
- ROE 的驱动因素是否与商业模式匹配？
- 变化趋势是否合理？
- 是否存在异常？

### 第五步：风险识别
- 净利率下降：竞争加剧、成本上升
- 周转率下降：应收账款增加、存货积压
- 杠杆率上升：债务风险增加

## 注意事项

1. **不同行业不可直接对比**：要在同行业内对比
2. **关注趋势而非绝对值**：变化方向比数值本身更重要
3. **结合定性分析**：数字背后的商业逻辑更重要
4. **警惕异常值**：突然的大幅变化需要深入调查
5. **现金流验证**：高 ROE 但现金流差可能有问题
