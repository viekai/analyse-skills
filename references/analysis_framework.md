# 投资分析框架

## 分析目标

为投资者提供全面、客观、深入的公司分析，帮助做出明智的投资决策。

## 核心原则

1. **客观性**：基于数据和事实，避免主观臆断
2. **全面性**：覆盖行业、商业模式、财务、风险等多个维度
3. **深度性**：不仅看表面数字，更要理解背后的商业逻辑
4. **实用性**：提供可操作的投资建议

## 分析框架

### 1. 行业分析（自上而下）

**目的**：理解公司所处的外部环境

**关键问题**：
- 这是一个好行业吗？（市场空间、增长性）
- 行业竞争激烈吗？（竞争格局）
- 公司在行业中的位置如何？（市场地位）

**分析要点**：
- 行业市场规模（TAM）
- 行业增长率（CAGR）
- 行业集中度（CR5、HHI）
- 行业驱动因素
- 行业周期性
- 公司市场份额和排名

### 2. 商业模式分析（核心）

**目的**：理解公司如何赚钱，为什么能赚钱

**关键问题**：
- 公司为客户创造什么价值？
- 公司如何获取收入？
- 公司的成本结构如何？
- 公司有什么竞争壁垒？

**分析框架**：
```
价值主张 → 客户细分 → 收入模式 → 成本结构 → 竞争壁垒
```

**竞争壁垒类型**：
- **技术壁垒**：专利、技术领先
- **品牌壁垒**：品牌认知、客户忠诚度
- **规模经济**：成本优势
- **网络效应**：用户越多价值越大
- **转换成本**：客户切换成本高
- **监管壁垒**：牌照、资质

### 3. 财务分析（杜邦框架）

**目的**：量化评估公司的盈利能力和财务健康度

**核心指标**：ROE（股东权益回报率）

**分析框架**：
```
ROE = 净利率 × 资产周转率 × 权益乘数
```

**分析步骤**：
1. 计算 ROE 及三因素（5年数据）
2. 与行业对比
3. 分析趋势变化
4. 结合商业模式解释
5. 识别风险信号

**详细指标**：
- **盈利能力**：毛利率、净利率、ROE、ROIC
- **运营效率**：应收账款周转、存货周转、资产周转
- **财务杠杆**：资产负债率、有息负债率、流动比率
- **现金流**：经营现金流、自由现金流、现金流/净利润
- **成长性**：营收增长率、利润增长率

### 4. 风险分析

**目的**：识别可能影响投资回报的风险因素

**财务风险**：
- **偿债能力风险**：
  - 流动比率 < 1：短期偿债压力
  - 速动比率 < 0.8：流动性风险
  - 资产负债率 > 70%：高杠杆风险
- **现金流风险**：
  - 经营现金流 < 净利润：利润质量问题
  - 自由现金流为负：需要持续融资
- **应收账款风险**：
  - 应收账款/营收 > 30%：回款压力
  - 应收账款增速 > 营收增速：可能虚增收入
- **商誉风险**：
  - 商誉/净资产 > 30%：减值风险
- **债务到期风险**：
  - 短期借款占比高：再融资风险

**市场情绪风险**（基于雪球等社交媒体）：
- 负面舆论
- 争议性话题
- 投资者担忧的问题

### 5. 估值分析

**目的**：判断当前股价是否合理

**估值方法**：
- **相对估值**：PE、PB、PS、EV/EBITDA
- **绝对估值**：DCF（折现现金流）

**分析要点**：
- 当前估值水平
- 历史估值区间
- 同行业估值对比
- 估值合理性判断

### 6. 投资建议

**目的**：提供可操作的投资建议

**建议框架**：
```
投资逻辑 → 风险收益比 → 操作策略 → 关注要点
```

**投资逻辑**：
- 核心看点（3-5个要点）
- 支撑逻辑
- 预期收益

**风险收益比**：
- 上行空间
- 下行风险
- 风险收益比评估

**操作策略**：
- 建议仓位
- 买入时机
- 止损止盈

## 分析流程

### 第一步：快速扫描
- 公司是做什么的？
- 所属行业如何？
- 财务数据是否健康？

### 第二步：深度研究
- 行业深度分析
- 商业模式拆解
- 财务数据详细分析

### 第三步：风险评估
- 识别所有风险
- 评估风险严重程度
- 判断风险可控性

### 第四步：估值判断
- 计算合理估值
- 判断安全边际

### 第五步：形成结论
- 投资逻辑
- 投资建议
- 关注要点

## 报告撰写要求

### 1. 结构清晰
- 使用标题层级
- 逻辑连贯
- 重点突出

### 2. 数据准确
- 引用数据要有来源
- 计算要准确
- 使用表格展示关键数据

### 3. 分析深入
- 不仅列数据，更要解释原因
- 结合商业模式分析财务数据
- 识别异常和风险

### 4. 客观中立
- 避免过度乐观或悲观
- 既要看到优势，也要看到风险
- 提供平衡的观点

### 5. 可读性强
- 使用简洁的语言
- 避免过多专业术语
- 重要结论要突出

## 常见陷阱

1. **只看财务数据，不看商业模式**：数字背后的逻辑更重要
2. **只看增长，不看质量**：高增长可能是虚假繁荣
3. **只看优势，忽视风险**：风险往往被低估
4. **只看当前，不看趋势**：趋势比绝对值更重要
5. **只看公司，不看行业**：行业天花板限制公司成长
